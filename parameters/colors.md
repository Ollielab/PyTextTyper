# colors

{% hint style="info" %}
'colors' from PyTextTyper, selection of colors to use with color(). Options include BLACK, RED, GREEN, ORANGE, BLUE, PURPLE, CYAN, LIGHTGREY, DARKGREY, LIGHTRED, LIGHTGREEN, YELLOW, LIGHTBLUE, PINK, LIGHTCYAN, RESET.
{% endhint %}

## Example:

```python
import pytexttyper

color(colors.GREEN)
```
